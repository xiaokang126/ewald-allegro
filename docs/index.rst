Ewald-Allegro
=============

Ewald-Allegro is an E(3)-equivariant machine-learning interatomic potential with **O(N) Ewald long-range electrostatic interactions**.
It extends the strictly local `Allegro <https://github.com/mir-group/allegro>`_ model with long-range physics.

The underlying Allegro library is implemented as a `NequIP <https://github.com/mir-group/nequip>`_ extension package.
